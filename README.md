### My Blog [Jekyll](http://jekyllrb.com/docs/configuration/)

> 搭建博客，并将静态部署到 github pages， 也能动态部署到 Server。
> bundle exec jekyll serve
> http://localhost:4000

    /demo
    　　　　|--　_config.yml `设置文件`
    　　　　|--　_layouts    `存放模板文件`
    　　　　|　　|--　default.html  `作为 Blog 的默认模板`
    　　　　|--　_posts      `存放 Blog 文章` 
    　　　　|　　|--　2012-08-25-hello-world.html  `年-月-日-文章标题.后缀名`
    　　　　|--　index.html  `首页`
