区别:
1. select 中 ng-repeat 写在 option 里， ng-options 是自动生成 option。
2. ng-options 一定和 ng-model 搭配使用。
3. ng-repeat 的 value 值是 string， 当是 number 时， 下拉列表失败。

4. ng-options 的 value 值是 string 或者 number。
